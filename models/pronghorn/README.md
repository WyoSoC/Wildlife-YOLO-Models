# Model Card — Pronghorn (*Antilocapra americana*)

Single-class YOLO11s detection model for pronghorn antelope,
trained primarily on NACTI camera-trap images.

## Training Data

| Source | Images | Notes |
|---|---|---|
| NACTI (LILA BC) | ~690,000 | 17.34% of full NACTI dataset; species-labelled, sparse bbox |
| WCS Camera Traps | supplementary | HuggingFace mirror |

Bboxes for NACTI images without annotations were generated by MegaDetector v5.

## Training Details

| Property | Value |
|---|---|
| Architecture | YOLO11s |
| Input size | 640 × 640 |
| Epochs | 100 |
| Augmentation | degrees=10°, scale=0.5, mosaic=1.0 |
| Device | CUDA |

## Known Limitations

- Camera-trap images are predominantly side-on; frontal/rear views have lower recall.
- At night (IR / low-light) the model performs on par with MegaDetector (not species-specific).
